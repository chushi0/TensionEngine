# Tension Engine

![build|coding](https://img.shields.io/badge/build-coding-yellowgreen)

Tension Engine 是一款轻量级跨平台游戏引擎。Tension Engine 使用 C++ 编写，使用 CMake 工具构建。您可以选择在代码中链接静态库或动态库使用。

[API 文档](wiki/api/Home.md)

| :------------: | :----: |
|  TensionCore   | 核心库 |
|  TensionAudio  | 音频库 |
|  TensionInput  | 输入库 |
| TensionNetwork | 网络库 |
